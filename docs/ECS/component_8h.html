◆ component_t

typedef struct component_s component_t

It's the structure of a component. A component is a part of an entity use to describe what is the entity and what it is able to do. Only systems can update component data. Components are only stores in entities. A component contains a type and some data.

Note
To create a new component, you have to create a new type and a new structure to store data.
Good practice is to create a constructor, a getter and a setter functions for your component.
