typedef struct system_s system_t

It's the structure of a system. A system is used to update a type of component or resource. It contains a type, a boolean and a function pointer. Systems are called by the world to be run.

Note
To create a new system, you have to create a new type and a new function to run the system.
Good practice is to create a constructor and use it to bind the function pointer to the run function.
